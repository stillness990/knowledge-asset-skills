# Knowledge Asset Skills

A Claude Code skill for turning tasks, technical problems, and projects into reusable, structured, searchable, and actionable knowledge assets.

## What It Does

This skill guides Claude to produce outputs such as:

- Knowledge notes
- SOPs and runbooks
- Architecture documents
- Project plans
- Troubleshooting guides
- Checklists

## Skill Structure

```text
.
├── SKILL.md
├── README.md
└── templates/
    ├── architecture.md
    ├── knowledge-note.md
    ├── project-plan.md
    ├── sop.md
    └── troubleshooting.md
```

## Usage

Install or copy this directory into your Claude Code skills directory, then invoke the skill by name:

```text
/knowledge-asset
```

Use it when you want responses to be structured as durable knowledge assets rather than one-off chat answers.

## Templates

The `templates/` directory contains reusable Markdown templates for common knowledge asset types.
